# math_6_checkup_tasks
Математический тренажёр для 6 класса – интерактивное приложение на Python с GUI (PyQt5), помогающее ученикам отрабатывать ключевые темы. Приложение генерирует случайные задания с проверкой ответов и виртуальной клавиатурой.  Тэги (Tags): python, pyqt5, math-trainer, education, desktop-application, mathematics, 6th-grade
